# AGENTS.md - AI Secretary 正本ルール

このフォルダは AI Secretary 本体の正本です。

## 正本

- 正本フォルダは `ai_secretary_v01`
- AI Secretary 本体として扱う
- 別プロジェクトの機能を混ぜない
- 他フォルダや別アプリを勝手に編集しない

## 現在状態

- 現在の管理状態は v6.3
- v6.0 以降でアプリ風UIへ整理済み
- PWA対応済み
- アプリ風ホーム画面あり
- 秘書キャラ表示あり
- 高級感のあるダークテーマあり
- 下部タブUIあり
- 機能詳細は専用ページへ分離する方針

## 起動

```powershell
python app.py
```

表示確認:

```text
http://127.0.0.1:5000/
```

iPhone確認用:

```text
http://<PCのIPアドレス>:5000/
```

## 優先方針

- iPhone表示を最優先
- アプリ風UIを優先
- プレゼント版品質を意識する
- デザイン重視
- スマホで押しやすいボタンにする
- ホームは見た目と操作感を優先する
- 長いフォームや一覧はホームに直接並べない

## 禁止事項

- 既存DBを勝手に変更しない
- 既存機能を削除しない
- Google連携を必須にしない
- OpenAI連携を必須にしない
- 外部APIなしでも起動できる状態を守る
- 別プロジェクトを混ぜない
- AI Secretary以外の正本を勝手に触らない
- 完成済みの動作を未確認で壊さない

## 開発方針

- 1チャット1機能を基本にする
- 小さく直す
- 既存リンク先を勝手に変えない
- 既存DB構造を変える場合は事前に確認する
- 保存する
- 構文チェックする
- 起動確認する
- 変更内容と確認結果を短く報告する

## 作業後チェック

必ず以下を行う。

1. 保存
2. `python -m py_compile app.py`
3. `python app.py` 起動確認
4. `http://127.0.0.1:5000/` 表示確認
5. 修正箇所、内容、確認結果を報告

## デザイン方針

- 高級感のあるAI秘書アプリにする
- ダークテーマを基本にする
- Glassmorphism風の表現を使う
- 余白を取りすぎず、iPhoneで1画面に収まりやすくする
- 下部タブは維持する
- ホーム画面はピタッと止まる操作感を優先する
- 専用ページは必要に応じてスクロール可

## 現在の主な構成

- `app.py` Flask本体
- `secretary.db` SQLite DB
- `templates/index.html` メイン画面
- `templates/components/home.html` ホームとタブ画面
- `templates/feature_page.html` 機能専用ページ
- `static/style.css` 画面デザイン
- `static/app_logo.png` 差し替え用ロゴ
- `static/secretary_character.png` 差し替え用秘書キャラ画像
